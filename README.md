<h1>Depth Crawler</h1>
<p>is a text based, dungeon crawler, action RPG with a simple looting and combat system as well as a fully realised story reliant on NPC dialogue and a quest system. This was a solo project developped over the span of two weeks in parallel 
with another project <a href="#">Death Delt Hand</a>. This game relies heavily on random procedural generation which allows for a different experience each and every playthrough, which affectsa the map, enemies and lootable items as well as other factors.</p>
<br>
<h2>Gameplay</h2>
<p>Primary Gameplay can be broken down into 4 main mechanics. These are:</p>
<ul>
  <li><p>Navigation</p></li>
  <li><p>Combat</p></li>
  <li><p>Looting</p></li>
  <li><p>Inventory</p></li>
</ul>
